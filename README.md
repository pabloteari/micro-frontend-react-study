# micro-frontend-react-study

micro-frontend is a project to study the resources of the micro-front end

## Installation

Use the package manager [npm] to install all micro.

```bash
#run in /container, /microFrontOne and /microFrontTwo
npm install
```

## Usage

```bash

#run in /container, /microFrontOne and /microFrontTwo
npm start
```
